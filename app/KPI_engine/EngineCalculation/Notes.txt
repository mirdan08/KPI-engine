- Si possono usare le funzioni di base
- Non si possono far eseguire funzioni prima di averle create (questo perchè bisognerebbe fare la parte di Checking ed esecuzione insieme / scritto apparte e non ha senso)
- Si possono scrivere nomi di altre funzioni (perchè ridanno un valore non bisogna passargli una variabile)
- Si possono creare gl' alert
- Non si possono  richiamare alert da altre funzioni / alert (non devi fare if else o cose simili)
- Tutte le base function devono ridare un float.
- Se esiste una KPI con lo stesso nome di un alert o di una function, verrà considerata la function / alert (se non è nostro compito controllare se sono disponibili le KPI)

- Viene controllato se ci sono operazioni che dividono per 0
